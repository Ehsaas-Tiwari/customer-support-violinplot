# Customer Analytics: Response Time Distribution by Support Channel

This project was created for Osinski Group to visualize customer support efficiency
using a professional-grade Seaborn violinplot.

## Business Objective
To analyze and visualize the distribution of customer response times across multiple
support channels for executive-level decision-making.

## Files Included
- `chart.py` → Python script that generates the Seaborn violinplot
- `chart.png` → Output visualization (512×512 pixels)
- `README.md` → Project documentation

## Visualization Type
- Seaborn Violinplot
- Synthetic business-realistic dataset
- Professionally styled using Seaborn best practices
